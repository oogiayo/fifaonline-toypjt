# 1. DB Parsing



## 1.1 선수 고유 식별자(spid) 메타데이터 조회



- 메타정보

###### API 정보

선수 고유 식별자(spid) 메타데이터를 조회합니다

선수 고유 식별자는 시즌아이디 (seasonid) 3자리 + 선수아이디 (pid) 6자리로 구성되어 있습니다.
시즌아이디는 / metadata/seasonid API로 조회할 수 있습니다.
